For now, the components are designed to show placeholder icons when images are not available.

## Image Requirements
- Format: JPG or PNG
- Aspect ratio: 16:9 (recommended)
- Resolution: At least 1200x675px for high-quality display
- File size: Optimized for web (under 500KB each)

## Adding Images
Simply place your caravan images in this directory with the exact filenames referenced in the caravan data structure in `/src/lib/getAllCaravans.js`.
